================================================================================
  🚀 PRÓXIMOS PASSOS (RÁPIDO)
================================================================================

1. COPIAR ARQUIVOS PARA SEU PROJETO
   
   Seus arquivos estão em: /mnt/user-data/outputs/
   
   Copie para seu projeto local (C:\PGM\sistema-relatorios):
   
   ✓ templates/ (pasta inteira) → root do projeto
   ✓ .gitignore → root
   ✓ Procfile → root
   ✓ app_init.py → app/__init__.py
   ✓ app_routes.py → app/routes.py
   ✓ app_models.py → app/models.py

2. TESTAR LOCALMENTE
   
   cd C:\PGM\sistema-relatorios
   venv\Scripts\activate
   pip install -r requirements.txt
   python run.py
   
   Abra: http://localhost:5000/painel?token=pgm-contenciosa-2026

3. CRIAR REPOSITÓRIO NO GITHUB
   
   git init
   git add .
   git commit -m "Initial: dashboard e estrutura básica"
   git remote add origin https://github.com/SEU_USUARIO/pgm-relatorios.git
   git push -u origin main

4. DEPLOY NO RENDER
   
   ⇒ Acesse: https://render.com
   ⇒ New Web Service → GitHub
   ⇒ Build: pip install -r requirements.txt
   ⇒ Start: gunicorn run:app --bind 0.0.0.0:$PORT
   ⇒ Adicione variables: SECRET_KEY, ACCESS_TOKEN, ENABLE_SCHEDULER

5. PRONTO!
   
   Seu sistema estará em: https://pgm-relatorios.onrender.com/painel?token=...
